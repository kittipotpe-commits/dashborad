# Simple Dashboard (Thai data)

วิธีรันบนเครื่องของคุณ:

1. ติดตั้ง dependencies
```bash
npm install
```

2. รันเซิร์ฟเวอร์พัฒนา
```bash
npm run dev
```

3. เปิดเบราว์เซอร์ที่ http://localhost:5173

หมายเหตุ:
- ผมตั้งให้แสดงสกุลเป็น THB (รูปแบบไทย) ใน KPI card
- ถ้าต้องการให้ผม deploy เป็นลิงก์สาธารณะ (Vercel / Netlify / Codesandbox) ผมสามารถช่วยสร้าง repo และ deploy ให้ได้ — แต่ผมต้องชื่อ owner/repo หรือสิทธิ์ในการ push (ให้คุณบอกชื่อ repo หรืออนุญาต)
